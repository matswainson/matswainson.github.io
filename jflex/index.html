jFlex

touch-device friendly jQuery carousel plugin

Download Github

Browser Support

  • Android 4.1+
  • Chrome 31+
  • Firefox 38+
  • IE 9+
  • Opera 30+
  • Opera Mini

How to use

  • <div class="flex">
        <ul class="slides">
            <li data-title="Slide 1">
                Slide 1
            </li>
            <li data-title="Slide 2">
                Slide 2
            </li>
            <li data-title="Slide 3">
                Slide 3
            </li>
        </ul>
    </div>
  • $('.flex').jFlex({
        autoplay: true,
        timing: 5000,
        titles: 'bottom'
    });

Options

  • autoplay true, false
  • fx none, slide
  • showArrows true, false
  • showPlayPause true, false
  • titles bottom, top
  • timing (ms)number (5000)
