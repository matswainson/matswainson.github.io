jFlex

touch-device friendly jQuery carousel plugin

Download Github

Options

  • autoplay true, false
  • customClass add an optional CSS class
  • highlightColor default: #555
  • showArrows true, false
  • showPlayPause true, false
  • swipeable true, false
  • theme light, dark
  • titles bottom, top
  • timing (ms)number (5000)

Browser Support

  • Android 4.1+
  • Chrome 31+
  • Firefox 38+
  • IE 9+
  • Opera 30+
  • Safari iOS 7.1+
  • Safari 8+
  • Opera Mini

How to use

  • <div class="flex">
        <ul class="slides">
            <li data-title="Slide 1">
                Slide 1
            </li>
            <li data-title="Slide 2">
                Slide 2
            </li>
            <li data-title="Slide 3">
                Slide 3
            </li>
        </ul>
    </div>
  • $('.flex').jFlex({
        autoplay: true,
        highlightColor: '#3b808d',
        showArrows: true,
        theme: 'dark',
        timing: 5000,
        titles: 'bottom'
    });
